## 🧠 What This Project Does

**Content Muse AI** helps creators generate fresh and engaging content ideas using Google's Gemini AI model.

The bot asks users:
- 🎯 Who their target audience is
- 📱 Which platform they are creating content for

Then it generates:
- Instagram post ideas
- YouTube video ideas
- Blog article topics
- Content calendar suggestions
- Creative marketing content
